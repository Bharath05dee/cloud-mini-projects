# Cloud & IT Mini Projects

## 1. Cloud Storage Setup and File Management

**Platform:** AWS S3 / Azure Blob Storage

### Description

This project focuses on understanding basic cloud storage concepts. A
cloud storage account was created, and files were uploaded and managed
with proper access permissions.

### Key Learnings

-   Cloud storage concepts
-   Public and private access permissions
-   Scalability and availability
-   Pay-as-you-use pricing model
Platform: AWS S3 / Azure Blob Storage
